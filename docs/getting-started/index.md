# Getting Started

Welcome to the deployment guide for My Kubernetes Homelab Platform. This section will
walk you through everything you need to know to get your homelab up and running.

## Overview

The deployment process is designed to be straightforward and automated. You'll need:

1. A running k3s cluster
2. Basic knowledge of Kubernetes and kubectl
3. Git for version control
4. About 30 minutes for initial setup

## What's Next?

Follow these guides in order:

1. [**Prerequisites**](prerequisites.md) - Set up your environment
2. [**Installation**](installation.md) - Deploy the platform
3. [**Scripts Reference**](scripts-reference.md) - Understand available automation scripts

!!! tip "Quick Start"
    If you already have k3s installed and configured, you can jump directly to the [Installation](installation.md) guide.
